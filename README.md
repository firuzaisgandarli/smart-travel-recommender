# 🌍 Smart Travel Recommendation System

An interactive travel recommendation dashboard built using **Python, Streamlit, Data Analytics, and Machine Learning concepts**.

This project analyzes real-world YouTube travel video metadata and generates destination recommendations by combining engagement metrics, content-based features, and user preference signals.

---

# 📌 Project Overview

The system explores travel-related video data and transforms it into meaningful recommendations and analytical insights.

The recommendation engine combines:

- 📊 Engagement-based metrics
- 🌿 Content-based travel interests
- 🧠 Machine learning probability scores
- 🎯 User preference filtering
- ⏱️ Video duration signals

The final result is an interactive dashboard where users can explore destinations, compare trends, and analyze travel behavior patterns.

---

# 🚀 Main Features

## 🌍 Smart Recommendation Dashboard
- Destination recommendation system
- Region-based filtering
- Interest-based filtering
- Country and city ranking
- Recommendation scoring

## 📊 Insights Dashboard
- Travel trend analysis
- Engagement analytics
- Country performance analysis
- City-level insights
- Content theme analysis

## ⚖️ Comparative Analysis Dashboard
- Budget vs Luxury comparison
- Food vs Nature comparison
- Vlog vs Non-vlog analysis
- Human presence vs non-human content comparison
